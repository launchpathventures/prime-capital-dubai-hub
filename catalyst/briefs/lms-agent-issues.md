After creating all issues, label them:

```bash
# If using GitHub CLI
gh issue edit 1 --add-label "copilot,lms-content"

# ... etc
```

Or add labels in the GitHub UI:
- `copilot` — Signals for Copilot assignment
- `lms-content` — Groups all LMS content issues
